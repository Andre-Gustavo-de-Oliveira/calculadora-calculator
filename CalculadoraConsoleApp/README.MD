# Calculadora

## Projeto

Desenvolvido durante o curso da [Academia do Programador](https://www.academiadoprogramador.net) Back-end 2026

## Introdução

Calculadora Contendo operações básicas a nível de console. As operações executáveis são as Seguintes: Soma, Subtração, Multiplicação e divisão. Outras funcionalidades extras são: Executar a TABUADA de Qualquer número escolhendo os valores de ínicio e término e também é possível ver o retrospecto das suas operações!

## Funcionalidade

- **Operações básicas** : Soma, subtração, multiplicação e divisão de maneira simples e direta!
- **Histórico de operações** : É possível verificar as últimas operações realizadas pela calculadora, assim não é difícil confirmar suas ações.
- **TABUADA** : É possível fazer a lista da tabuada de qualquer número, sendo possível personalizar o intervalo, assim poupando de verificar números desnecessários! Simples assim você consegue retirar as suas dúvidas!

## Como utilizar o programa

1. Clone ou baixe os arquivos do repositório.

2. Abra o seu emulador de terminal de preferência, e navegue até a pasta raiz projeto baixado!

3. Utilize o comando abaixo para restaurar as dependências do projeto

    ```
    dotnet restore
    ```

4. Em seguida Compile e rode o projeto com o comando:

    ```
    dotnet run --project CalculadoraConsoleApp
    ```

5. Ao executar o código no seu computador, escolha entre uma das opções válidas pelo **MENU** de acesso.
Caso escolha alguma das operações aritmética básicas, Insira dois valores e o resultado irá aparecer em sua tela! Ao pressionar a tecla **ENTER** você voltará ao **MENU** principal, podendo sair do programa ou escolher outra opção. Caso escolha o Histórico de operações ele irá mostrar suas últimas contas, caso não tenha efetuado nenhuma ainda o resultado em branco irá aparecer solicitando que volte ao **MENU** principal e escolha outra opção. Por fim caso escolha a Tabuada, será solicitado o número que será multiplicado e após isso será solicitado os multiplicadores de sua preferência, Após a execução será possível retornar ao **MENU**. Lembrando que a ordem de execução é totalmente exclusiva dos seus comandos! Fique a vontade para experimentar todas as possibilidades!!
